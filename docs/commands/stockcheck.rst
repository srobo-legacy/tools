stockcheck
==========

Synopsis
--------

``sr stockcheck [-h] [-<n>] <schematic> [[-<n>] <schematic> ...]``

Description
-----------

Perform a stock check on a number of schematics.

Options
-------

--help, -h
    Display help and exit.

Examples
--------

.. code::

    $ sr stockcheck -100 servo-v4-hw/*.sch
