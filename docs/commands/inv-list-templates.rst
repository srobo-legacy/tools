inv-list-templates
==================

Synopsis
--------

``sr inv-list-templates [-h] [--parts] [--assemblies]``

Description
-----------

List templates in the inventory.

Options
-------

--help, -h
    Display help and exit.

--parts, -p
    Show part templates.

--assemblies, -a
    Show assembly templates.

Examples
--------

.. code::

    $ sr inv-list-templates --assemblies
    box-9l-rub
    box-3l-rub
