cam-serial
==========

Synopsis
--------

``sr cam-serial [-h]``

Description
-----------

Display the serial number of all the SR-compatible cameras attached to the
host.

The SR-compatible cameras are the Logitech *C500* and *C270*.

Options
-------

--help, -h
    Display help and exit.

Examples
--------

.. code::

    $ sr cam-serial
    dah297ssh9fj03
    $
