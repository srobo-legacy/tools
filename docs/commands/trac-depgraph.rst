trac-depgraph
=============

Synopsis
--------

``sr trac-depgraph [-h] <ticket>``

Description
-----------

Create a dependency graph for a particular ticket.

Options
-------

--help, -h
    Display help and exit.

--summaries, -s
    Show ticket summaries in the graph.

Examples
--------

.. code::

    $ sr trac-depgraph 1000
