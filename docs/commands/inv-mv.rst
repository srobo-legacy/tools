inv-mv
======

Synopsis
--------

``sr inv-mv [-h] [-a] <asset_code> [<asset_code> ...]``

Description
-----------

Move an asset into the current working directory.

Options
-------

--help, -h
    Display help and exit.

--assy, -a
    If the asset is part of an assembly, move the whole assembly rather than
    just the asset.

Examples
--------

.. code::

    $ sr inv-mv 000
