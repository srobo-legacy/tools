ide-list-teams
==============

Synopsis
--------

``sr ide-list-teams [-h] [--server <server>]``

Description
-----------

List the teams in the IDE.

Options
-------

--help, -h
    Display help and exit.

--server <server>, -s <server>
    The server running the IDE to connect to. This defaults to the Student
    Robotics server.

Examples
--------

.. code::

    $ sr ide-list-teams
