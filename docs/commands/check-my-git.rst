check-my-git
============

Synopsis
--------

``sr check-my-git [-h]``

Description
-----------

Check that Git is correctly configured.

Options
-------

--help, -h
    Display help and exit.

Examples
--------

.. code::

    $ sr check-my-git
    Your git is correctly configured. :)

    $ sr-check-my-git
    You need to tell git who you are.
    Run `git config --global user.name Your Name`.
