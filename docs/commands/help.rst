help
====

Synopsis
--------

``sr help [-h] <command> [<command> ...]``

Description
-----------

Look up commands in the SR tools documentation.

Options
-------

--help, -h
    Display help and exit.

Examples
--------

.. code::

    $ sr help inv-new-asset

Caveats
-------

This currently relies on an internet connection to open up ``readthedocs.org``.
