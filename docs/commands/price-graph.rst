price-graph
===========

Synopsis
--------

``sr price-graph [-h] [-<n>] <schematic> [[-<n>] <schematic> ...]``

Description
-----------

Create a price graph for multiple schematics.

Options
-------

--help, -h
    Display help and exit.

Examples
--------

.. code::

    $ sr price-graph -100 servo-v4-hw/servo-4-output.sch
